# Archive

Thư mục này chứa ghi chú, checklist cũ hoặc tài liệu không còn là tài liệu vận hành hằng ngày. Không xóa hoàn toàn để giữ lịch sử dự án.

Nếu chuyển file vào archive:

- cập nhật link trong `README.md` và `docs/README.md`,
- chỉ archive khi nội dung không còn cần đọc thường xuyên,
- không archive security/migration/production docs đang được dùng.
